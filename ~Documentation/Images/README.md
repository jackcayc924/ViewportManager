# Image Assets for ViewportManager

This directory contains all promotional and documentation images for the ViewportManager plugin.

## Directory Structure

```
~Documentation/Images/
├── Features/          # Feature showcase images (Layout Designer, Quick Start, etc.)
├── Screenshots/       # In-game/in-editor screenshots
├── Diagrams/         # Technical diagrams and flowcharts
└── UI/               # UI mockups and widget examples
```

## Image Guidelines

### Naming Convention
- Use descriptive names: `layout-designer-interface.png`
- Include version if needed: `quick-start-wizard-v1.0.png`
- Use lowercase with hyphens

### Recommended Sizes
- **Feature Images**: 1920x1080 or 1280x720
- **Screenshots**: Native resolution (1080p+ recommended)
- **Diagrams**: 1200x800 minimum
- **UI Examples**: Actual viewport size

### File Formats
- **PNG**: UI screenshots, diagrams (supports transparency)
- **JPG**: Photos, large screenshots (smaller file size)
- **GIF**: Short animations (< 5MB)

## Usage in README

Reference images using relative paths:

```markdown
![Layout Designer](~Documentation/Images/Features/layout-designer.png)

![Quick Start Wizard](~Documentation/Images/Screenshots/quick-start-wizard.png)
```
